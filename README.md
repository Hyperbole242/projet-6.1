# 💇‍♀️ Coiffure Congo – Site vitrine

Un site vitrine de 3 pages (Accueil, Services, Contact) réalisé dans le cadre du projet **6.1 – Développement Web** à Akieni Academy.  
Ce projet met en avant un salon de coiffure avec un design moderne et responsive.

---

## 🚀 Fonctionnalités

- 🌐 **3 pages HTML valides** : Accueil, Services, Contact
- 🎨 **Design premium** : dégradés animés, glassmorphism, effets 3D au hover
- 📱 **Responsive** : adapté aux mobiles, tablettes et ordinateurs
- 🖼️ **Sections riches** :
  - Accueil : Hero avec image de fond + CTA
  - Services : Cards avec icônes, avis clients, section tarifs
  - Contact : Formulaire stylisé avec icônes + infos pratiques
- ⚡ **Animations au scroll** pour dynamiser la navigation
- 🔗 **Navigation cohérente** entre les pages
- 📄 **README documenté** et déploiement sur GitHub Pages

---

## 🛠️ Technologies utilisées

- **HTML5** : structure sémantique
- **CSS3** : flexbox, grid, animations, media queries
- **Git/GitHub** : gestion de versions et déploiement
- **GitHub Pages** : hébergement gratuit du site

## 📱 Responsivité

- Grilles `grid-template-columns: repeat(auto-fit, minmax(...))`
- Formulaire centré avec `max-width`
- Media queries pour ajuster navigation et Hero
- Boutons et champs adaptés au tactile
